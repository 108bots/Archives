'allheaders.tpl' contains the stardard SIP header definitions. Alter this file to add more headers.

DON't delete or change the name of the 'allheaders.tpl' file.

Though a rules file can have multiple rules as in 'fuzzTestRules.xml'. Currently, rules file should have only one rule. This  limitiation is from the SSTF wizard point of view. Here each rule file corresponds to a single template file (.tpl) which in turn corresponds to a single message type.
