
The folder contains the following files:
----------------------------------------

C Source Files:
---------------
stdfs.c

Headerfiles:
------------
mydeclarations.h

Help file:
---------
Help.txt : contains the command description and usage. (used by the stdfs program)


Makefile: Use this to compile the C source files
---------

Other files:
------------
exp.txt : Gives a brief explanation of the entire filesystem, the design, implementation
          commands supported and so on.

How to execute:
---------------
Run "make" command. Once the executables are generated, run as shown below:

stdfs
        "Usage: stdfs <return>"


Login:
-----
  Initially no user will be created. So create user first. So commands can be executed only by
"root", so create a the "root" as well.
